# SIMULATIONS AND OUTPUTS

# CASE 1
* WHEN DISTANCE <= 100 
* Servo 180 degree rotates
* LCD prints close tap

# CASE 2 
* WHEN DISTANCE > 100 
* Servo 90 degree rotates 
* LCD prints open tap

# CASE 3
* WHEN DISTANCE > 300 (max)
* Servo stops rotating
* Both LEDS turn ON
* LCD displays tank is full.
